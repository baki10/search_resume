**Демонстрационное приложение собирает резюме с сайта hh.ru и сохраняет их в базу данных.**
--
**Используемые фреймворки и библиотеки:**
 
- spring mvc
- jsoup
- jpa (hibernate 5)
- PostgreSQL
- angularJS
 

_**db.properties**_ 
-файл со свойствами БД 

**Сборка проекта:** 
**`mvn clean install`**

**Запуск:**
**`mvn jetty:run`**

_**http://localhost:8080/**_
